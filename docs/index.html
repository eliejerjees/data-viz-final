Big 5 Player Radar

Pick any two players from Europe's Top 5 leagues and see how they match up across goals, assists, passing, defending, and more. Every stat is scored 0–100 relative to the full dataset — so a striker and a centre-back can be compared on the same chart fairly. Two players are preloaded to get you started; swap in anyone you like using the search boxes below.

How to use ▾
  • Filter by league — use the league tabs to narrow the player pool to one competition.
  • Search for players — type a name in either search box and select from the dropdown.
  • Filter by position — tap FW · MF · DF · GK under a search box to narrow results.
  • Focus a radar — click a polygon or legend label to lock it; then hover the dots for exact stats and a head-to-head diff.
  • Rankings — clicking the radar opens a ranked attribute card for that player.
  • Swap axes — click any axis label on the chart to replace it with a stat from the full catalog.
  • 0–100 scores — the concentric rings mark 20 · 40 · 60 · 80 · 100; a score of 100 is the dataset best for that stat.
  • Radar presets — use the Attacking · Midfield · Defending · Goalkeeping tabs to switch the whole axis group at once.
